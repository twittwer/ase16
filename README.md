# Setup 

### Prerequisites
- Docker
- Docker-Compose
- Node
- NPM

1. Clone the repository
´
	git clone https://github.com/twittwer/ase16.git
	cd ase16
´

2. Install Dependencies
´
	cd ./backend && npm install
	cd ../frontend && npm install 
´

3. Start Containers
´
	docker-compose up
´
